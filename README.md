# Social Network

## Description:
This repo contains all the backend code needed for a social network web application using a NoSQL database! The use of NoSQL allows for large amounts of unstructured data. The application is set up for users to share their thoughts, react to friendsâ€™ thoughts, create a friends list, and update or delete any of these components as desired.
